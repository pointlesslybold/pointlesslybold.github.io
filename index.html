Web design, programming and more

Some example projects to show my strengths in web design and web development. This list will continue to expand over time

Mouse Race Web Game

A web game built on canvas using pure Javascript, without the use of any frameworks or libraries. Supports variable screen sizes/aspect ratios and touch controls on mobile.

Burger Shop

A mobile-first web page made without using any frameworks or libraries. Has flexible responsive design and a minimalistic interactive slideshow widget, which can be easily expanded with more slides and supports a variety of content sizes.

Product Page

Responsive mobile-first landing page featuring many interactive elements, such as tabbed content, sliding content, expandable menus on mobile, as well as eye-catching gallery and functional FAQ block. Uses flexbox for layout, minimal amount of javascript and smooth CSS animations.

Minesweeper for Web

Web version of the classic game "Minesweeper" written in pure JS. Features include custom grids, play area scaling and hint system. Supports variety of window sizes from 360-pixels-wide and up.
