Web design, programming and more

Some example projects to show my strengths in web design and web development. This list will continue to expand over time

Mouse Race

A simple game based on canvas, fully adaptive to any aspect ratio and resolution, also supports keyboard and touchscreen controls. Written in pure JS.

Mouse Race Backend

A simple Node.JS-based application powering score table for Mouse Race. Uses SQLite and Express framework.

Product Page

Responsive mobile-first landing page featuring many interactive elements, such as tabbed content, sliding content, expandable menus on mobile, as well as eye-catching gallery and functional FAQ block. Uses flexbox for layout, minimal amount of javascript and fluid CSS animations.

Minesweeper for Web

Web version of the classic game "Minesweeper" written in pure JS. Features include custom grids, play area scaling and hint system. Supports variety of window sizes from 360-pixels-wide and up.
